# P01 - QIIME 2 Demultiplexing: dysautonomia dataset 

Feb 10th, 2023

## Purpose:To import and sort the 16S rRNA sequences based on barcode information in QIIME2 

## Material: Materials: macOS Mojave 10.14.6 with QIIME2 (v2021.11)

## Method:
1. Connect to Cisco Anyconnect Secure VPN. Open the terminal and login to MICB 475 class server using the provided login credentials.
2. Create a dedicated directory for project 2: /data/team6_dysautonomia_data
3. Import and demultiplex the dysautonomia dataset (path: /mnt/datasets/project_2/dysautonomia) to create a qza file with
demultiplexed samples (output path: data/team6_dysautonomia_data/team6_dysautonomia_paired_demux_seqs.qza).
5. Create a visualization for the demultiplexed samples (output is demux.qzv).
6. Move the demux.qzv file to your local computer directory and view it using the QIIME 2
